## btptf export

Export specific resources from an SAP BTP subaccount

```
btptf export [flags]
```

### Options

```
  -o, --config-dir string   folder for config generation (default "generated_configurations")
  -h, --help                help for export
  -s, --subaccount string   Id of the subaccount
```
